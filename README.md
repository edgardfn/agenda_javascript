# agenda_javascript

Agenda digital com Javascript:

Adicionar pelo Javascript a tarefa digitada no campo input, no dia da semana escolhido no select, a hora também é exibida no campo da semana correspondente.

Ao clicar no botão Criar Tarefa é verificado se o campo tarefa está vazio, caso sim é retornado mensagem de alerta até usuário entrar com algum conteúdo no campo.

Botão limpa tarefas realiza a limpeza de todos as tarefas que foram inseridas.

Responsividade para Iphone 6/7/8.
